==============
extension.icon
==============


Extension: Icon ViewHelper
==========================

Outputs the icon of the extension key. Supports both
extension key and extension name arguments.

Arguments
=========


.. _extension.icon_extensionname:

extensionName
-------------

:aspect:`DataType`
   string

:aspect:`Required`
   false
:aspect:`Description`
   Name, in UpperCamelCase, of the extension to be checked
